Golden Eagle Flight Visualization

A data-driven visualization project exploring eagle flight motion through 3D reconstruction, trajectory analysis, and visual storytelling.

View Repository

Project Overview

We transform high-resolution bio-logging data into a dynamic 3D visualization of golden eagle flight. By combining GPS, accelerometer, gyroscope, and magnetometer data, we reconstruct real flight trajectories and body orientation.

The result is an interactive system that maps pitch, roll, and yaw onto a 3D eagle model, enabling a deeper understanding of flight dynamics and animal behavior.

Pipeline

1. Data Input

Flight or angle data is loaded from structured files such as CSV.

2. Processing

Orientation and motion values are prepared for mapping onto the model.

3. Visualization

The eagle model is animated to produce a readable visual flight sequence.
